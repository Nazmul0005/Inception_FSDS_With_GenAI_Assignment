# Python Conditional Statements - Assignment Collection

A collection of 10 beginner-friendly Python programs demonstrating conditional statements (if-elif-else) through practical real-world scenarios.

## Programs Included

1. **Temperature Check** - Categorizes temperature as cold, cool, or hot
2. **ATM Withdrawal System** - Validates withdrawal against account balance
3. **Student Grade Calculator** - Assigns letter grades based on marks
4. **Bus Ticket Discount** - Calculates ticket discounts by age group
5. **Restaurant Bill with Discount** - Applies 10% discount for bills over 1000
6. **Login Authentication** - Validates username and password credentials
7. **Traffic Light System** - Interprets traffic light colors
8. **Leap Year Checker** - Determines if a year is a leap year
9. **Job Eligibility Checker** - Checks age and qualification requirements
10. **Electricity Bill Calculator** - Calculates bills based on unit consumption

## How to Run

1. Open the Jupyter notebook in Google Colab or locally
2. Run each cell individually to test different programs
3. Enter values when prompted by the program

## Requirements

- Python 3.x
- Jupyter Notebook or Google Colab

## Learning Objectives

- Understanding if-elif-else conditional statements
- Working with user input
- Implementing decision-making logic
- Basic arithmetic operations and string comparisons
